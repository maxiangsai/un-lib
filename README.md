# xt-lib

## 安装依赖

### 全局

```bash
$ yarn
// or
$ npm i
```

### 子模块

```bash
$ yarn bootstrap
# or
$ npm run bootstrap
```

## 启动服务

```bash
$ npm start
# or
$ yarn start
```

## 打包

### 静态资源打包

```bash
$ npm run docs:build
# or
$ yarn docs:build
```

### 组件及资源打包

```bash
$ npm run build
# or
$ yarn build
```
