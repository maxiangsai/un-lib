## 轻松上手

### @xt/components

```shell
npm install @xt/components --save --registry=http://nexus-op.vipthink.cn/repository/npm-group
```

或者

```shell
yarn add @xt/components --registry=http://nexus-op.vipthink.cn/repository/npm-group
```
